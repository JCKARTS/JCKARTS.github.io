HTML HEAD ELEMENTS

"Head Elements encapsulate all the elements that are not seen by the viewer"

Html lang="en"

This Tag allows you to state the HTML document that the browser is going to read is based in English.

Meta Charset="UTF-8"

The charset attribute specifies the character encoding for the HTML document.

Link rel="stylesheet" src="./styles.css"

Link elements within the head element help connect your CSS file sheet with the HTML code.

Head <>

Always encapsulate all the head elements within the Head Tag so that elements can be imployed.

All other font links...

All other font libraries can also be linked in the head element, then emploed either through inline styles or CSS elements.

HTML Body Elements

"Body elements are everything seen by the viewer on the browser."

Div

Div elements are short for division. These help break parts of the page into seperate objects.

Span

Span elements are meant for short pieces of text, that are not a paragraph. So that you may highlight or turn into a link.

EM

Encapsulating text with the EM elements would make the text it encapsulated italic.

Strong

Encapsulating text with the Strong element tags would bolden all the text it encapsulates.

BR

BR elements when wrapped before or after an element add a page break to the content displayed.

CSS Text Selectors

CSS text and font selectors allow you to modify the way text is represented on your webiste

font-family:Helvetica;

The font-family selector allows you to be able to modify and change the typeface of the content presented on your page.

text-decoration: *;

Text-decoration selector allows you to modify the type of decoration presented around the content on your page.

font-size: *;

The font-size command allows you the ability to dicatate the size of the text information on the page.

Letter-spacing:4px;

This selector allows you to control the amount of space between letters in content on your page.

word-spacing: 3px;

This selector allows you to control the spacing between words in your content on the page.
